Pero... ¿Qué es un sistema operativo?
El sistema operativo es la pieza fundamental del software para ejecutar tareas en cualquier dispositivo inteligente. La mayoría de los dispositivos tienen dos modos de operación: modo kernel y modo usuario. El sistema operativo se ejecuta en modo kernel. En este modo, el sistema operativo tiene acceso completo a todo el hardware y puede ejecutar cualquier instrucción que la máquina sea capaz de hacer. El resto del software se ejecuta en modo usuario, en el cual sólo un subconjunto de las instrucciones de máquina es permitido y durante un tiempo limitado.
El núcleo o kernel en particular, se ocupa de las instrucciones que afectan el control de la máquina, Proporcionan una asignación ordenada y controlada de los procesadores, memorias y dispositivos de E/S, entre los diversos programas que compiten por estos recursos, haciendo abstracción de los complejos conjuntos de hardware.
Un concepto clave en todos los sistemas operativos es El proceso. Un proceso es en esencia un programa en ejecución. Cada proceso tiene asociado un espacio de direcciones, que es una lista de ubicaciones de memoria. Toda la información acerca de cada proceso (además del contenido de su propio espacio de direcciones) se almacena en una tabla del sistema operativo, conocida como la tabla de procesos, la que es un arreglo de estructuras, una para cada proceso que tenga. Así, un proceso es listado de direcciones que guarda el contenido de sus registros y los pasos necesarios para reiniciarlo cuando sea llamado.
Cuando escribimos una instrucción en un (dir, mount por ejemplo ) intérprete de comandos o shell de una terminal hacemos una llamada al sistema de administración de procesos. El proceso se “levanta” en memoria ejecutándose de acuerdo a su listado de instrucciones. Cuando termina ejecuta una llamada para terminarse a si mismo.
Otro concepto clave de casi todos los sistemas operativos es El sistema de archivos. una de las funciones principales del sistema operativo es ocultar las peculiaridades de los discos y demás dispositivos de E/S, presentando al programador un modelo abstracto limpio y agradable de archivos independientes del dispositivo. La mayoría de los sistemas operativos usan el concepto de un directorio como una manera de agrupar archivos. En el caso de Unix-Linux los archivos son “montados” para que el kernel sepa de su existencia así como el montar los discos físicos.

Katherine Bouman. La chica que ideó el algoritmo para ver un agujero negro
Observar objetos distantes es extremadamente difícil porque la resolución de un telescopio es insuficiente. Se necesitaría uno muy grande para captar más detalle. Para solucionar estos se usa una técnica llamada “interferometría”. Se hacen capturas de varios telescopios en diferentes lugares para formar una sola imagen.
Como están separados, es necesario reconstruir la imagen completa a partir de diferentes partes. Aquí es donde entra la genialidad de la Katherine Bouman como están separados alrededor de la Tierra, significa que también observan el objeto desde un ángulo distinto. Hay que corregir esas diferentes perspectivas cuando estés rearmando la imagen.
Pero como la Tierra rota esos ángulos no van a ser siempre iguales. Además, las diferentes posiciones de la tierra también significan que la misma onda de luz va llegar a diferentes tiempos a cada telescopio. Si la luz viaja en el vacío es fácil, pero llega a la atmósfera y ya no se propaga igual. Se defasa.
Hasta que Bouman se dió cuenta que utilizando tres referencias se puede hacer una corrección. Con todo esto hay una infinidad de imágenes que podrían resultar. ¿Como saber cual es la mas probable? Para ello los algoritmos deben “entrenarse” con probabilidades a priori de cómo debería verse un objeto astronómico sin influenciar el resultado.
Esas técnicas salen de probabilidades estadísticas bayesianas y la distribución de Gauss. Es decir, técnicas de machine learning (aprendizaje automático). El ruido de las imagines esta distribuido de forma gaussiana. La data recolectada son PETABYTES por todos los telescopios. Por lo que fue necesario enviar los discos por correo aéreo.
